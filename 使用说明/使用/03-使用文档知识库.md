# 使用文档知识库

在左侧点击 **知识库 → 文档知识库**。

![文档知识库](../images/标注/12-文档知识库.png)

操作顺序：

1. 点击 **新建文件夹**，按项目或资料类型建立分类。
2. 选中左侧文件夹，再点击 **上传文档**。
3. 等待解析进度达到 100%，文档状态变成“完成”。
4. 点击 **查看条目** 或 **查看 Markdown** 检查解析结果。
5. 状态为“失败”时，根据提示处理文件后重新解析。

超长文档会自动分段解析、匹配前后内容并补齐遗漏，无需手动拆分。知识库准备好后，可在技术方案的“目录生成”设置中选择它，让目录和正文参考已有资料。
