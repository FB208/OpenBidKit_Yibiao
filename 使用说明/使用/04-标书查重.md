# 标书查重

在左侧点击 **标书检查 → 标书查重**。

第一步上传文件：

- **招标文件**：可选，可以上传多份。
- **投标文件**：必选，需要上传多份互相比较。

多份投标文件都引用了招标文件中的相同内容时，不计为投标文件之间的重复。确认文件后点击 **下一步**。

![标书查重选择文件](../images/标注/13-标书查重选择文件.png)

等待四个维度全部显示“已完成”：

- **元数据**：比较设备、账号、作者、编辑时间等文件信息。
- **目录**：比较章节结构和标题顺序。
- **正文**：筛查重复段落、表格和关键描述。
- **图片**：通过图片哈希查找完全相同的原图；截图、压缩等相似图片可能无法识别。

![标书查重结果](../images/标注/14-标书查重结果.png)

在结果页切换顶部维度，查看完成数量、命中内容和涉及文件。修改原投标文件后，点击 **重新查重** 生成新结果。
