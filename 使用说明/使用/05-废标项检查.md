# 废标项检查

在左侧点击 **标书检查 → 废标项检查**。

操作顺序：

1. 选择招标文件和需要检查的投标文件。
2. 点击 **下一步**，等待软件提取无效投标和废标条款。
3. 检查提取结果，必要时补充自定义检查项。
4. 点击 **下一步 → 开始检查**，等待全部任务完成。

![废标项检查结果](../images/标注/15-废标项检查结果.png)

结果页包含三类检查，顶部均显示“已完成”后再查看：

- **废标项检查**：检查投标文件是否响应硬性要求。
- **错别字检查**：检查明显的文字错误。
- **逻辑谬误检查**：检查前后矛盾、时间不一致和参数冲突等问题。

结果可按投标文件筛选。展开问题后，可以查看原文与位置、谬误原因和修改建议；确认是误报时可删除该条。AI 检查结果只作为辅助，提交标书前仍需人工复核。
